A. 不同的社会有完全不同的道德

C. 不同的社会条件下有着不同的社会价值观念

D. 社会道德规范包括基本道德规范和一系列特定道德规范

```

知 识 点：第5章 > 第1节 > 道德 (理解)

教材页码：P155-158

试题难度：本题共被作答 28393 次 正确率 82 %
```

B. 绝大多数的道德规范都是以法律作为价值基础的

C. 法律的实施对道德观念的培养起到强化促进作用

```

试题难度：本题共被作答 42778 次 正确率 65 %

```

**27 [单选题]** 一种行为在法律上并未被禁止，但是往往被社会公众所诟病，这说明了道德与法律(       )。
